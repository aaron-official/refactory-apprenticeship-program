# broHOSTS (Media Streaming Platform)

broHOSTS is a comprehensive, mobile-responsive web application for discovering and streaming movies and TV shows.

---

## 1. Cinematic Discovery & Browsing

**The Experience:** A dark, high-end "Netflix-style" interface designed for effortless visual browsing.

- User opens the app and is greeted by a massive, high-definition Hero Banner with rotating movie highlights.
- User scrolls down to see curated content rows such as "Trending Now" or "Top Rated."
- User hovers over a poster to trigger a subtle scale effect, making the title visually pop.
- User clicks on a poster to proceed to that title's details page.

---

## 2. Live Interactive Search

**The Experience:** An instant search results grid that responds as you type, without ever leaving the page.

- User clicks the magnifying glass icon in the sidebar to open the search interface.
- User begins typing a title (e.g., "Batman") into the search field.
- A live result grid populates the screen in real-time, displaying matching movie and TV posters instantly.
- User clicks a result to transition directly to that title's details page.

---

## 3. TV Show Season & Episode Navigation

**The Experience:** A deep-dive layout that makes exploring long-running series feel organized and intuitive.

- User clicks a TV show poster to land on its dedicated page with a cinematic background.
- User views the Season List displayed on the left side of the screen.
- User selects a season, which updates the Episode Grid on the right with thumbnails and descriptions.
- User types in the episode search bar to find a specific episode title.
- User clicks the episode to begin watching it immediately.

---

## 4. Personal Collection (Watchlist)

**The Experience:** A dedicated space to save "must-watch" content for later viewing.

- User finds a movie or show they want to save and clicks the bold "Add to Watchlist" button.
- The button visually changes to a "Remove" state, confirming the title has been saved.
- User clicks "Watchlist" in the sidebar to navigate to their personal collection.
- A personalized grid of saved posters is displayed for the user to browse at any time.

---

## 5. Unified Login & Signup Flow

**The Experience:** A smooth, overlay-based entry that keeps the user in the "streaming" mindset.

- User clicks the "Sign In" or "Create Account" button to trigger a sleek, dark-themed overlay window.
- User enters their details or selects a social login icon to authenticate.
- Upon success, the overlay disappears and the Watchlist and Profile icons in the sidebar automatically unlock for full access.

# broRacks (Mobile Money Payment Gateway)

broRacks is a dedicated payment gateway REST API and dashboard that abstracts complex local telco payments (like MTN and Airtel) for developers.

---

## 1. Dashboard Overview

**The Experience:** A high-performance, dark-themed cockpit featuring glowing neon green stats and real-time wallet balances that feel like a cinematic fintech control room.

- User enters their secret credentials on the dark login screen.
- User clicks the "Sign In" button.
- The interface fades into a sleek, black command center.
- Glowing green numbers appear at the top showing the "Available Balance."
- A scrolling list of the most recent money movements slides into view at the bottom.

---

## 2. Requesting Payments

**The Experience:** A clean, focused interface designed for speed, using helpful blue alerts and instant feedback loops to confirm money requests are on their way.

- User clicks the "Collect" icon in the left-hand menu.
- User types the customer's phone number and the requested amount into the form.
- User clicks the bright "Initiate Collection" button.
- A green "Success" box appears showing the exact fee and the final amount to be received.
- The customer's physical phone screen lights up with a prompt to approve the payment.

---

## 3. Withdrawals

**The Experience:** A secure and professional portal for sending funds to mobile phones, featuring clear fee breakdowns and high-contrast status badges.

- User clicks the "Withdraw" link in the navigation sidebar.
- User enters the recipient's phone number and the amount to send.
- User clicks the "Confirm Transfer" button.
- The system subtracts the amount from the "Available Balance" immediately.
- A new row appears in the history table with an orange "Pending" badge that turns green once successful.

---

## 4. Activity Audit

**The Experience:** An immersive, detailed archive of every financial event, presented in a crisp, filterable table with high-visibility color coding.

- User clicks the "Transactions" list in the sidebar.
- A full-screen table appears showing every past collection and withdrawal.
- User hovers their mouse over a specific row to highlight a transaction.
- User clicks the row to reveal a hidden panel of extra details.
- The panel slides out to show the exact second the money moved and its unique tracking ID.

---

## 5. Security Setup

**The Experience:** A minimalist, technical space for managing "secret keys" and notification links, emphasizing security with clean layouts.

- User clicks the "API Keys" icon in the navigation menu.
- User clicks the "Generate" button to create a new access code.
- A masked code (hidden by dots) appears in a new row.
- User clicks a "Reveal" icon to see the full secret code.
- User clicks the "Copy" button to save the code for use in their own external business software.

# ocs4dev (Fintech API Integration Assistant)

ocs4dev is a specialized AI partner designed to help developers seamlessly build and troubleshoot payment gateways—like mobile money platforms and card networks—through an interactive, visual chat interface.

---

## 1. The Interactive Chat Interface

**The Experience:** A clean, expansive main dialogue window designed for deep focus, welcoming the user with friendly placeholder suggestions.

- User opens the main screen and views the large chat area.
- User locates the wide text box fixed at the bottom of the screen.
- User types a question or prompt into the text field.
- User clicks the prominent "Send" button next to the text box.
- The interface smoothly streams the assistant's response into a new message bubble above.

---

## 2. AI Assistant Selection Menu

**The Experience:** A neatly organized, professional sidebar panel that gives the user easy, visual toggle controls over their preferred smart assistant.

- User looks to the dedicated settings panel on the left side of the screen.
- User locates the "Model Provider" options block.
- User clicks a circular toggle button to select a preferred brand name.
- User moves to the "Tier" section immediately below.
- User clicks a toggle to choose between a faster or smarter response level.

---

## 3. Secure Passcode Entry

**The Experience:** A secure, collapsible drawer accented with subtle warning highlights to ensure peace of mind when handling sensitive text.

- User scrolls down the left sidebar menu.
- User clicks an expandable heading marked with a small key icon.
- A highlighted security warning box appears on the screen.
- User clicks into one of the password-protected text fields.
- User types the secret text, watching as the interface masks the characters for privacy.

---

## 4. Quick-Start Suggestion Grid

**The Experience:** A tidy, inviting row of clickable, pill-shaped buttons sitting just beneath the chat box to spark instant inspiration.

- User looks just below the main text input field.
- User scans the grid of pre-written question buttons.
- User clicks on a specific question button that catches the eye.
- The selected text instantly populates the main input box.
- The system automatically submits the text and begins generating a visual response.

---

## 5. Formatted Instruction Blocks

**The Experience:** A highly readable, beautifully structured message stream where conversational text is clearly separated from neatly boxed, color-coded instructions.

- User observes the completed message bubble in the main chat area.
- User reads the introductory conversational text with bolded headers.
- User scrolls down to a distinctly separated, dark-themed box within the message.
- User reads the specially colored, monospaced text inside the box.
- User highlights the contents of the dark box to copy it for personal use.
